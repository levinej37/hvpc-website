⚠  Punchlist Item — Google Photos Setup Required

Connect Google Photos albums before launch

To activate this gallery:

  1. Create a Google account for the parish (or use an existing one)
  2. Upload photos into Google Photos, organized into albums (e.g. "Church & Grounds", "Feast Days", "Parish Life")
  3. Open each album → click Share → Get shareable link → copy the album URL
  4. Replace the GOOGLE_PHOTOS_ALBUM_URL placeholders below in this HTML file with the actual album URLs
  5. Delete this notice box

The embed iframes are already wired up — just paste in the URLs and everything works automatically.
